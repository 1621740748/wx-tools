Wx-tools是基于微信公众平台API的轻量级框架。
基于Wx-tools你可以开速开发一个订阅号/服务号的web应用后台。

## 特性：
* 统一、简单的API，可以快速上手。
* 链式赋值编程，更加容易理解和使用。
* 对于微信服务器发来的消息，提供匹配器（Matcher），拦截器（interceptor），处理器（Handler）接口，定制实现，具有可扩展性。
